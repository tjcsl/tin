###########
Add Numbers
###########

----------
Assignment
----------
Write a program that has a function called ``add_num`` that takes two parameters
:math:`x` and :math:`y`, and returns their sum :math:`x+y`.


----------------
Example Solution
----------------

.. code-block:: python

    def add_num(x, y):
        return x + y
